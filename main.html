Time and Space Complexity

Time Complexity is used to measure how much time an algorithm takes to execute as the input size increases.It helps in analyzing the performance and efficiency of an algorithm. By studying time complexity, developers can identify whether a solution is efficient or not. An efficient algorithm reduces the execution time and improves the overall performance of a program.

Space Complexity: O(n) means the memory used by the algorithm increases linearly with the input size (n). The algorithm requires extra memory storage such as arrays or lists, so the memory usage grows proportionally with the input.
